How to Apply CSS to HTML

To add CSS into an HTML file, we can use the "<link>" tag. We use the "rel" attribute to declare the relationship between the linked document and the HTML file, in this case: rel="stylesheet". We can use the "src" attribute to write the specific location, or URL, of the CSS file. From here, we can then create a separate file titled "style.css" and write all of our CSS code there.

Code Example:

<link rel="stylesheet" href="/style.css" />

How to Use JavaScript in a Webpage

If you want to run a JavaScript file on multiple pages in a web site, the best practice is to create an external JavaScript file, instead of writing the same code repeatedly. Save the external file with a ".js" extension, and then refer to it using the "src" attribute in the "script" tag. Dont forget to include the ending "/script" tag!

Code Example:

<script src="myscripts.js"></script>
